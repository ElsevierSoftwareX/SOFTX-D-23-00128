The Speech Group has several pieces of software that you can use for analyzing and synthesizing speech, doing math and scientific calculations, text processing, and graphics on UNIX systems.

Speech tools:

  • xkl
    • This is the principal speech analysis and synthesis program, allowing real-time analysis of waveforms, spectrograms, and a wide variety of spectra.  Uses Klwav files.
  • syn
    • Command-line version of the Klatt formant speech synthesis program, driven by 63 acoustic phonetic parameters -- the synthesis engine in several commercial products, including DECtalk and Sensimetrics HLSyn.   Saves synthesized speech as a Klwav file.  Algorithm details can be found in J. Acoust. Soc. Am. 67(3), 1980, and J. Acoust. Soc. Am. 87(2), 1990.
  • lspecto
    • A batch-style program for producing postscript plots of the waveform, spectrogram, formants, and pitch. Also capable of producing a synthesizer doc file, to make speech copying easier.
  • concat
    • Concatenate two or more Klwav files into one Klwav file.
  • klgain
    • Scale Klwav audio file.
  • klinfo
    • Find out information (e.g. sampling rate) about a Klwav file.
  • klplay
    • Command-line program to play a Klwav file.
  • klrecord
    •  Command-line program to record a Klwav file.
  • raw2kl and kl2raw
    • Convert a raw (binary, 16-bit, linear) data to a Klwav file.  Both take a "-swap" option for doing byte-swapping.
  • mat2kl.m and kl2mat.m
    • Routines you can use in matlab to convert between Matlab and Klwav files.
  • mac2kl and kl2mac
    • Command-line programs to convert between a Mac audio file (AIFF) and Klwav files.  mac2kl is useful when you transfer files that you recorded on the Mac and want to use them in xkl.  Similarly, if you want to use Klwav files for playing sounds on the Mac (e.g. perceptual experiments using Psyscope) , you can use kl2mac.
  • ms2kl and kl2ms
    • Command-line programs to convert between Microsoft (RIFF) .wav files and Klwav (.wav) files.
  • sd2kl and kl2sd
    • Command-line programs to convert between ESPS (SD) .sd files and Klwav (.wav) files.   Both take a "-swap" option for doing byte-swapping.
  • sox
    • Conversion utility.  Sox translates sound files from one format to another, possibly doing adding a sound effect or sampling rate conversion.
  • matlab
    • Matlab is a matrix-crunching language with powerful graphics capabilities.
  • xwaves/esps
    • xwaves is an interactive program for display and manipulation of time-series data.  It lets you, among other things, browse through lengthy signals, select portions to examine features in detail, cut and paste segments of the signal, do interactive spectral analysis and signal  annotation, play back signals, compute and display speech formant frequencies and pitch contours, and perform other editing and display operations.  Operations can be selected from menus with the mouse, activated by single keystrokes at the keyboard, or invoked through commands in an interpreted language.
    • ESPS are command-line programs and C functions and procedures for performing various speech analyses.

Math/Scientific tools:

  • matlab
  • gnuplot

Text processing:

  • Latex (available on all machines)

Graphics:

  • pstoedit (available on septum, palate, formant)
    • This program allows conversion of a Postscript image (e.g. a spectrum created in xkl) to an editable format hat can be used by xfig.
    • Usage:    pstoedit  -f fig  spectrum.ps  spectrum.fig
  • xfig
    • This is a menu-driven tool that allows the user to draw and manipulate objects, similar to MacDraw and IslandDraw.
